## 本次 browser-use 操作摘要

1. 使用 UI 菜单执行“退出登录”，再次访问 `/dashboard`，确认路由守卫回到 `/login`。
2. 使用默认账号登录，进入 `/dashboard`。
3. 通过 DOM 快照确认工作台 V2 信息结构和空数据状态。
4. 点击顶部“快速排课”，确认弹窗标题和表单字段。
5. 在窄视口中滚动到“快速操作”区，分别点击“快速调课”“新增学生”“录入收款”，确认 URL 和目标页面文本。
