状态：已完成。

已知当前学生端仍保留移动端优先的基础结构和旧版 Tab 入口，后续需要先确认新版学生端范围，再决定是渐进改造还是重新梳理页面主线。

2026-04-28 已启动学生端新版承接的基础工程收口：

- 将学生端主页面和详情页改为路由级懒加载，避免所有页面进入首包。
- 配置 Vite 手动拆分 React 与图表依赖，构建产物从单个约 1.2 MB 入口包拆成多个小页面包和共享依赖包。
- `npm run build` 通过，超 500 kB chunk 警告消失；仍保留 Vite CJS Node API deprecation warning。
- 新增学生端首页，聚合账户余额/余额风险、最近课程、待完成作业和最新反馈。
- 新增学生端账户明细页，展示当前余额、最近扣费和最近收款。
- 新增 `GET /api/billing/my/account` 学生/家长端只读账户接口，并补充 pytest 回归。
