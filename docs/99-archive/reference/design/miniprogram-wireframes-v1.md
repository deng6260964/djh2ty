# 家教辅助系统 - 微信小程序端线框图

> 状态：归档
> 范围：微信小程序
> 更新：2026-04-26
> 说明：小程序端线框归档参考；老师端 V2 是当前主要产品基线，学生/家长端后续改造需重新校准。

> 约定：以下线框图以 750rpx 设计稿宽度描述，模拟手机竖屏界面（约 375px CSS宽度）。
> 每个页面示意图宽度约 40 字符，代表手机屏幕内容区域。
> 底部 TabBar 固定在所有主页面底部。

```
+--------------------------------------+
|                                      |
|                                      |
|                                      |
|         [系统 Logo 图标 96rpx]       |
|          家教辅助系统                 |
|          学生专属学习平台             |
|                                      |
|                                      |
|                                      |
|   +--------------------------------+ |
|   |   [微信头像占位圆]  微信昵称  | |
|   |   授权后即可查看你的学习信息  | |
|   +--------------------------------+ |
|                                      |
|   [    微信一键授权登录    ]         |
|    按钮：绿色 #07C160，圆角 44px     |
|    高度：88rpx，字号：34rpx，白色    |
|                                      |
|                                      |
|   老师已添加你的账号后才能登录       |
|   （灰色说明文字，居中）             |
|                                      |
|                                      |
+--------------------------------------+
```

**授权流程**
1. 用户点击按钮 → 弹出微信官方授权弹窗（获取头像、昵称）
2. 用户同意 → 调用 `wx.login()` 获取 code → 发送到后端换取 openid
3. 后端验证 openid 是否在学生列表中：
   - 已绑定：登录成功，跳转课程 Tab
   - 未绑定：显示提示「你的微信尚未绑定学生账号，请联系老师」
4. 首次绑定成功后，token 存入本地 storage，后续静默登录

## 通用小程序页面规范

### 导航栏设计

```
+--------------------------------------+
|  [< 返回]     页面标题        [...] |
|  16rpx        居中 34rpx Bold  可选 |
+--------------------------------------+
```
- 背景色：`#FFFFFF`
- 标题字号：34rpx，字重 600，颜色 `#111827`
- 返回按钮：系统默认（微信提供）
- 自定义导航栏（需要时）：总高度 = 状态栏高度 + 44px

### 通用卡片样式

```
+----------------------------------+
|                                  |  margin: 24rpx 32rpx
|  内容                            |  padding: 32rpx
|                                  |  background: #FFFFFF
+----------------------------------+  border-radius: 16rpx
                                       box-shadow: 0 2rpx 8rpx rgba(0,0,0,0.06)
```

### 列表项样式

```
+--------------------------------------+
| [图标]  主标题                   [>] |
|         副标题/说明文字              |
+--------------------------------------+  高度约 96rpx，底部分隔线
```

### 空状态设计

+--------------------------------------+
