# 家教辅助系统 - 产品需求说明

## 一、项目背景

一名独立家教老师，需要一套工具来管理自己的日常教学工作。
目前主要通过微信和手动记录来管理学生信息、课程安排、作业、收费等，效率低、容易遗漏。
需要一个简单、实用的系统，帮助老师提升管理效率，同时让学生和家长能方便地查看相关信息。

## 二、目标用户

| 角色 | 说明 |
|------|------|
| 老师（管理员）| 系统的主要使用者，负责所有管理操作 |
| 学生 | 通过微信小程序查看课程、作业、反馈、进度、资料 |
| 家长 | 通过微信小程序查看子女的学习情况 |

> 当前阶段：单老师模式（一个账号管理所有内容），预留多老师扩展接口

## 三、系统组成

| 端 | 说明 |
|----|------|
| 管理端（Web）| 老师在电脑/手机浏览器上使用，进行所有管理操作 |
| 学生端（微信小程序）| 学生和家长在微信内使用，查看相关信息 |

## 四、核心功能模块

### 4.1 学生管理
- 添加、编辑、删除学生信息
- 学生基本信息：姓名、年级、科目、联系方式（家长手机）、备注
- 查看学生详情：历史课程、作业完成情况、成绩趋势
- 搜索学生（按姓名/科目/年级）

### 4.2 排课管理
- 为学生安排课程（一对一）
- 课程信息：学生、科目、上课时间（日期+时间段）、时长、备注
- 上课时间冲突检测（同一时间不能安排两个学生）
- 日历视图查看课程安排
- 标记课程状态：待上课、已完成、已取消

### 4.3 作业管理
- 布置作业：选择学生（可多选）、截止日期、作业内容（支持富文本）
- 查看作业完成情况
- 批改作业：评分、评语
- 统计：各学生作业完成率、提交情况

### 4.4 课堂反馈
- 每节课结束后记录课堂反馈
- 反馈内容：本节课表现、知识点掌握情况、存在问题、下节课计划
- 支持反馈模板（快速填写常用内容）
- 推送给学生/家长（通过微信消息通知）

### 4.5 学习进度跟踪
- 记录每次课/考试的成绩
- 知识点掌握情况标记（已掌握/学习中/待学习）
- 生成学习报告（可导出 PDF）
- 成绩趋势可视化图表

### 4.6 收费管理
- 设置各科目课时单价
- 自动计算收费（按实际上课课时）
- 记录收款（现金/转账）
- 查看收费汇总报表（按学生/按时间段）
- 欠费提醒

### 4.7 资料管理
- 上传教学资料（文档、图片、PDF等，单文件 ≤50MB）
- 分类管理（按科目/年级/类型）
- 选择资料分享给指定学生
- 学生在小程序端查看/下载资料

### 4.8 考试辅导
- 管理真题库（按科目、年份、类型归类）
- 词汇库管理（外语科目）
- 创建模拟考试（从真题库随机抽取）
- 学生在小程序端做模拟题

### 4.9 通知管理
- 系统通知（作业提醒、课程提醒）
- 课前自动提醒（上课前 X 小时通知学生/家长）
- 手动发送通知

## 五、管理端功能页面

1. **登录页** - 账号密码登录
2. **首页（仪表盘）** - 今日课程、待处理作业、本月收入等数据概览
3. **学生管理** - 学生列表及详情
4. **课程管理** - 日历视图 + 列表视图
5. **作业管理** - 作业列表、布置、批改
6. **课堂反馈** - 反馈记录及新增
7. **学习进度** - 成绩图表、知识点地图
8. **收费管理** - 账单列表、收款记录
9. **资料管理** - 文件上传、分类、分享

## 六、小程序端功能页面

底部导航 5 个 Tab：
1. **课程** - 查看自己的课程安排（日历/列表）
2. **作业** - 查看待完成作业和已完成作业
3. **反馈** - 查看老师的课堂反馈
4. **进度** - 查看自己的成绩和学习报告
5. **资料** - 查看/下载老师分享的资料

## 七、非功能需求

| 类别 | 要求 |
|------|------|
| 部署环境 | 阿里云 2 核 2G 服务器 |
| 用户规模 | 单老师，最多管理 50 名学生 |
| 响应性能 | 普通操作 ≤ 1 秒，报告生成 ≤ 5 秒 |
| 文件存储 | 本地存储，总容量预算 ≤ 10GB |
| 微信集成 | 微信登录（小程序端）、消息推送 |
| 安全 | 账号密码登录，JWT 鉴权，数据不公开 |
| 可用性 | 24×7 可用，允许每月 1 次维护停机 |

## 八、优先级（MVP 核心功能）

P0（必须有）：
- 学生管理
- 排课管理（含冲突检测）
- 作业管理
- 课堂反馈
- 收费管理

P1（重要但可后续迭代）：
- 学习进度跟踪
- 资料管理
- 小程序端

P2（可选增强）：
- 考试辅导
- 通知推送
- PDF 导出报告
