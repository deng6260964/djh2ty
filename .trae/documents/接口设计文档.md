# 英语家教教学管理系统 - 接口设计文档

## 1. 接口概述

本文档定义了英语家教教学管理系统的所有API接口，包括用户认证、课程管理、作业管理、考试管理等模块。

**基础信息：**

* 基础URL：`http://localhost:5000/api`

* 认证方式：JWT Token

* 请求格式：JSON

* 响应格式：JSON

**通用响应格式：**

```json
{
  "success": true,
  "message": "操作成功",
  "data": {},
  "code": 200
}
```

## 2. 用户认证模块

### 2.1 用户注册

**接口地址：** `POST /auth/register`

**请求参数：**

| 参数名          | 类型     | 必需 | 描述                  |
| ------------ | ------ | -- | ------------------- |
| phone        | string | 是  | 手机号                 |
| password     | string | 是  | 密码（6-20位）           |
| name         | string | 是  | 姓名                  |
| role         | string | 是  | 角色（teacher/student） |
| invite\_code | string | 否  | 邀请码（学生注册必需）         |

**请求示例：**

```json
{
  "phone": "13800138000",
  "password": "123456",
  "name": "张老师",
  "role": "teacher"
}
```

**响应示例：**

```json
{
  "success": true,
  "message": "注册成功",
  "data": {
    "user_id": 1,
    "phone": "13800138000",
    "name": "张老师",
    "role": "teacher"
  }
}
```

### 2.4 题目管理相关

**创建题目**

**接口地址：** `POST /questions`

**请求参数：**

| 参数名 | 参数类型 | 是否必填 | 描述 |
|--------|----------|----------|------|
| course_id | int | 是 | 课程ID |
| category_id | int | 否 | 分类ID |
| type | string | 是 | 题目类型(single_choice/multiple_choice/fill_blank/essay/true_false) |
| content | string | 是 | 题目内容 |
| options | array | 否 | 选择题选项 |
| correct_answer | string | 是 | 正确答案 |
| difficulty | int | 是 | 难度等级(1-5) |
| explanation | string | 否 | 题目解析 |

**获取题目列表**

**接口地址：** `GET /questions`

**查询参数：**

| 参数名 | 参数类型 | 是否必填 | 描述 |
|--------|----------|----------|------|
| course_id | int | 是 | 课程ID |
| category_id | int | 否 | 分类ID |
| type | string | 否 | 题目类型 |
| difficulty | int | 否 | 难度等级 |
| page | int | 否 | 页码，默认1 |
| limit | int | 否 | 每页数量，默认20 |

**智能组卷**

**接口地址：** `POST /questions/auto-generate`

**请求参数：**

| 参数名 | 参数类型 | 是否必填 | 描述 |
|--------|----------|----------|------|
| course_id | int | 是 | 课程ID |
| total_questions | int | 是 | 题目总数 |
| difficulty_distribution | object | 是 | 难度分布配置 |
| type_distribution | object | 否 | 题型分布配置 |
| category_ids | array | 否 | 指定分类ID列表 |

**请求示例：**

```json
{
  "course_id": 1,
  "total_questions": 20,
  "difficulty_distribution": {
    "1": 0.2,
    "2": 0.3,
    "3": 0.3,
    "4": 0.15,
    "5": 0.05
  },
  "type_distribution": {
    "single_choice": 0.6,
    "multiple_choice": 0.2,
    "fill_blank": 0.2
  }
}
```

### 2.5 作业管理相关

**创建作业**
```
POST /api/assignments
```

请求参数：
| 参数名 | 参数类型 | 是否必填 | 描述 |
|--------|----------|----------|------|
| course_id | int | 是 | 课程ID |
| title | string | 是 | 作业标题 |
| description | string | 否 | 作业描述 |
| due_date | string | 是 | 截止时间(ISO格式) |
| question_ids | array | 否 | 手动选择的题目ID列表 |
| auto_generate | object | 否 | 自动生成配置 |

自动生成配置参数：
| 参数名 | 参数类型 | 是否必填 | 描述 |
|--------|----------|----------|------|
| total_questions | int | 是 | 题目总数 |

| type_distribution | object | 否 | 题型分布 |
| category_ids | array | 否 | 指定分类 |

```json
{
  "course_id": 1,

  "due_date": "2024-01-15T23:59:59Z",

}
```

```json
  }
}
```

## 9. 错误码定义

| 错误码 | 描述      |
| --- | ------- |
| 200 | 成功      |
| 400 | 请求参数错误  |
| 401 | 未授权访问   |
| 403 | 权限不足    |
| 404 | 资源不存在   |
| 409 | 资源冲突    |
| 422 | 数据验证失败  |
| 500 | 服务器内部错误 |

**错误响应格式：**

```json
{
  "success": false,
  "message": "手机号已存在",
  "code": 409,
  "errors": {
    "phone": ["该手机号已被注册"]
  }
}
```
